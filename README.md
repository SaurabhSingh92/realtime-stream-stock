# realtime-stream-stock

Another real-time project to work with finace data using yahoo finacne API and plot a graph able to consume the real time data. 

<b>Python library</b> - yfinance 
Will help us extract the stock detials on real time. 

<b> Matplotlib </b>
Plot and FuncAnimation is used to plot a graph will keep polling new data in-background and show updated values on the screen. 

<b> Streamlit</b>
Another library to implement real-time polt. (<in-progres>)
